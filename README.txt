# Catálogo de Animes

Aplicação web desenvolvida como projeto de TCC com foco em descoberta e recomendação de animes através de pesquisa interativa, filtros e visualização de dados.

---

## Funcionalidades

- Busca de animes pelo nome
- Busca de animes por filtros/gêneros
- Exibição de detalhes completos sobre os animes
- Visualização de dados através de gráficos interativos
- Calendário de lançamentos de animes
- Sistema de recomendação baseado no humor do usuário

---

## Tecnologias Utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript (ES6+)

### Bibliotecas
- Chart.js

### API
- API Jikan (MyAnimeList)

---

## API Utilizada

O projeto utiliza a API pública **Jikan**, responsável por fornecer dados da plataforma **MyAnimeList**, como:

- títulos
- avaliações
- imagens
- gêneros
- datas de lançamento
- popularidade
- informações gerais dos animes

## Objetivo do Projeto

O projeto foi desenvolvido com o objetivo de criar uma plataforma interativa de descoberta de animes, unindo pesquisa, recomendação personalizada e visualização de dados em uma interface moderna e intuitiva.
