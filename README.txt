# Catálogo de Animes

## Funções
 - Buscar conteúdo pelo nome
 - Busca de conteúdo por filtros
 - Exibição de detalhes sobre os animes
 - Visualização de dados através de gráficos interativos


## Tecnologias Utilizadas
# Frontend 
- HTML5
 - CSS3
 - JavaScript (ES6+)
 - Chart.js
 - API Jikan

# Backend
 - Python
 - Flask
 - SQLite
 - Flask-CORS

## API Utilizada
O projeto utiliza a API pública "Jikan", que fornece dados da plataforma "MyAnimeList".
