# Catálogo de Animes

## Funções
 - Buscar conteúdo pelo nome
 - Busca de conteúdo por filtros
 - Exibição de detalhes sobre os animes

## Tecnologias Utilizadas
 - HTML5
 - CSS3
 - JavaScript
 - API Jikan

## API Utilizada
O projeto utiliza a API pública "Jikan", que fornece dados da plataforma "MyAnimeList".
